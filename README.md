# 项目简介

## 核心功能
- 登录与注册
- 数据预览
- 年月时间分析
- 商品评分分析
- 订单评论数据分析
- 商品评论数据词云图
- 自然语言处理（NLP）评论情感分析
- LDA评论主题分析
- Bayes评论分类

## 适用场景
- 商品市场分析
- 客户评论监控
- 品牌形象评估
- 产品改进决策支持

## 技术特点
- 使用Python编程语言
- Flask框架搭建
- Pyecharts库进行数据可视化展示
